# サービス概要
歌い手とミックス師のためのマッチングアプリです。

歌い手様が、自身の予算や希望の納期に合致したミックス師様を見つけられることを目的としています。

検索ボックスに希望するミックス師様の情報を記載し、条件が合えばメッセージを送ってください。
